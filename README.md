# Nextjs fullstack boilerplate

## Build

```console
npm run build
```

## Develop

On first install:
```console
npm run setup 
````

Start dev server:

````console
npm run dev
````

### Bundle analizer

````console
npm run buindle:analyze
````

## Database migrations

Deploy in production:

````console
npm run db:migrate:prod
````

Migrate in development:

````console
npm run db:migrate:dev
````

## Features:

- Typescript
- Emotion
- Tailwind
 * Twin.macro
 * Flowbyte
* Prisma
* Next-ApiKit
